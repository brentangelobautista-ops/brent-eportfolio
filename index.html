About the Course

The course introduces students to the fundamentals of data management using the R programming language. It focuses on data infrastructure, organization, and management technologies that support business intelligence and enterprise decision-making. Students learn how to manage structured and unstructured datasets while maintaining accuracy, consistency, and transparency in enterprise data systems. The course also introduces RStudio commands and functions used for organizing, storing, and analyzing data in simulated business scenarios.

Learning Objectives

  • Understand the fundamentals of data management
  • Use the R programming language for statistical analysis
  • Perform exploratory data analysis (EDA)
  • Organize and manage datasets in enterprise contexts
  • Use RStudio tools and commands effectively

Coursework

Week 1 – Introduction to Data Management

Introduction to data management concepts and the role of data in enterprise systems.

Week 2 – R Fundamentals and Statistical Foundations

Installation of R and RStudio and introduction to statistical functions in R.

Week 3 – Exploratory Data Analysis (EDA)

Analyzing datasets, identifying patterns, and understanding distributions.

Week 4 – Univariate Data Analysis

Analyzing individual variables using descriptive statistics.

Week 5 – Bivariate Data Analysis

Understanding relationships between two variables using statistical methods.

Tools and Technologies

  • R Programming Language
  • RStudio
  • Statistical Analysis Methods
  • Data Visualization in R

Reflection

This course provided practical experience in managing and analyzing data using the R programming language. Through different exercises and laboratory activities, students gained insights into how data can be structured, processed, and interpreted to support decision-making in real-world business environments.
