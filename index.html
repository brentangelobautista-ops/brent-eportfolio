About the Course

The course Data Management and Statistical Analysis Using R introduces students to the basic concepts of data management and how data can be analyzed using the R programming language. It focuses on how organizations organize and manage data so that it can support business intelligence and decision-making.

Through this course, students are introduced to tools such as R and RStudio, which are used to explore datasets, perform statistical analysis, and interpret data. The lessons emphasize the importance of accurate, consistent, and meaningful data within enterprise systems.

Although the subject can be challenging, especially for students who are not very familiar with programming, it provides an important foundation for understanding how data analysis works in real-world business environments.

Learning Objectives

  • Understand the fundamentals of data management
  • Become familiar with the R programming language
  • Use RStudio for basic statistical analysis
  • Explore datasets using exploratory data analysis
  • Understand univariate and bivariate analysis
  • Recognize the importance of data in business decision making

My Learning Journey (Prelim Period)

Week 1 – Introduction to Data Management

During the first week, I learned about the importance of data management in organizations. I was introduced to how businesses rely on structured data systems to organize and store information.

Week 2 – Introduction to R and RStudio

This was my first time working with the R programming language and the RStudio environment. At first, it was confusing to understand the commands and how the system works.

Week 3 – Exploratory Data Analysis

I learned that before analyzing data deeply, analysts first explore the dataset to understand its structure and identify patterns.

Week 4 – Univariate Analysis

This topic focused on analyzing a single variable in a dataset using statistical measures such as mean and median.

Week 5 – Bivariate Analysis

I learned that two variables can be analyzed together to identify relationships or correlations between them.

Challenges I Encountered

One of the main challenges I experienced in this subject was learning how to use the R programming language. Since it involves coding and statistical concepts, it was not always easy for me to follow every command or function used during the activities.

Sometimes I needed more time to understand how certain operations work in RStudio. There were moments when I felt confused about the outputs or how the code affected the dataset.

Even though the subject was challenging, I still tried to follow the lessons and complete the activities as much as possible. Through practice, I slowly started to become more familiar with the environment.

Coursework Activities

Throughout the prelim period, we worked on different exercises that introduced us to RStudio and statistical analysis. These activities involved exploring datasets, running commands, and observing the results generated by the system.

Although I am still learning how to use the tools properly, the exercises helped me understand how analysts work with data and how information can be interpreted using statistical techniques.

Student Reflection

During the prelim period of this course, I realized that data management and statistical analysis are not easy subjects, especially for students who are still learning how programming works.

Personally, I found some parts of the course challenging, particularly when working with the R programming language. Understanding the commands and functions sometimes took more time for me, but I still tried to follow the lessons and practice whenever possible.

Even though I struggled with some topics, I was still able to gain a basic idea of how data can be explored and analyzed. The course helped me understand that data plays an important role in helping organizations make decisions.

I know that I still have many things to improve when it comes to statistical analysis and programming, but this course helped me become more aware of how data is handled in real-world business environments.

Moving forward, I hope to continue improving my understanding of the tools and concepts introduced in this course so that I can apply them better in the future.
